1. Loggar in på wordpress.org och markera det som 'favorit'
2. egen wordpress, 'lägg till tillägg' -> http://localhost:8080/wp-admin/plugin-install.php?tab=favorites 
3. tryck på 'hämta favoriter'
4. tryck på 'installera'
5. tryck på 'aktivera'
